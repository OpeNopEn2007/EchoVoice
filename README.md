# EchoVoice

> 多端轻量级 AI 语音输入法

打破传统语音输入"说啥是啥"的痛点，让用户自由表达，AI 智能润色不改变原意。

## 核心特性

- 🎯 **自由表达**：无需一口气说完美，边说边整理
- 🤖 **本地 AI**：纯本地运行，无需联网，保护隐私
- 📱 **多端支持**：iOS / Android / Windows / macOS / Linux
- ⚡ **轻量高效**：模型小巧，运行快速，内存占用低
- 🔒 **隐私优先**：语音数据永不离开设备

## 分端设计

### 移动端
- 智能悬浮球，不影响正常操作
- 长按悬浮球开始语音输入
- 保留原有输入法，无缝切换

### PC 端
- 后台静默运行
- 快捷键触发（默认 F9 / Capslock）
- 录音 → 润色 → 自动上屏

## 技术栈

- **移动端**：Flutter + Dart
- **桌面端**：Tauri + Rust
- **ASR**：Whisper.cpp（本地语音识别）
- **LLM**：llama.cpp（本地文本润色）
