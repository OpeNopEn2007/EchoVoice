## 模块职责

### 1. 音频录制 (audio)
- 使用 cpal 进行跨平台音频录制
- 支持 16kHz 采样率（Whisper要求）
- 实时音频流处理

### 2. ASR识别 (asr)
- 使用 whisper.cpp 本地语音识别
- 支持中英双语
- 模型：Whisper tiny-q5_0 (31MB)

### 3. LLM润色 (llm)
- 使用 llama.cpp 本地文本润色
- 模型：SmolLM2-360M (手机) / 1.7B (桌面)
- 提示词工程控制润色行为

### 4. 全局快捷键 (hotkey)
- 使用 global-hotkey 监听系统快捷键
- 支持 F9 / Capslock 触发

### 5. 系统托盘 (tray)
- 使用 tray-icon 提供系统托盘图标
- 设置入口和退出功能

## 多端策略

### 桌面端 (Tauri/Rust)
- 后台运行，无界面
- 全局快捷键触发
- 完整模型支持 (1.7B)

### 移动端 (Flutter)
- 悬浮球UI
- 长按录音
- 轻量模型 (360M)

## 模型配置

| 平台 | ASR模型 | LLM模型 | 总大小 | 内存占用 |
|------|---------|---------|--------|----------|
| 手机 | Whisper tiny (31MB) | SmolLM2-360M (250MB) | ~280MB | ~600MB |
| 桌面 | Whisper base (74MB) | SmolLM2-1.7B (1GB) | ~1.1GB | ~2.5GB |

## 隐私设计

- 所有模型本地运行
- 音频数据不离开设备
- 无云端API调用
- 可选：本地日志记录（可关闭）
