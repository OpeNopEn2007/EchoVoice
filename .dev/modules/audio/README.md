# Audio 模块

> 音频录制模块 - 负责麦克风音频采集

---

## 职责

提供跨平台的音频录制功能，支持开始/停止录音，返回标准格式的音频数据。

---

## 设计文档

- [详细设计](./design.md) - 架构、流程、技术细节
- [接口定义](./interface.md) - API 规范

---

## 关键特性

| 特性 | 说明 |
|------|------|
| 采样率 | 16000 Hz（Whisper 要求）|
| 格式 | f32 PCM |
| 通道 | 单声道 |
| 后端 | cpal（跨平台）|

---

## 依赖

- `cpal` - 跨平台音频 I/O
- `anyhow` - 错误处理
