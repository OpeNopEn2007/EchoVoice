# Config 模块

> 配置管理模块 - 管理用户设置和配置文件

---

## 职责

管理应用配置，支持用户自定义设置，提供配置的加载和保存功能。

---

## 设计文档

- [详细设计](./design.md) - 架构、流程、技术细节

---

## 关键特性

| 特性 | 说明 |
|------|------|
| 格式 | YAML |
| 位置 | ~/.config/echovoice/config.yaml |
| 加载 | 启动时自动加载 |
| 保存 | 修改后自动保存 |

---

## 依赖

- `serde` + `serde_yaml` - 序列化
- `dirs` - 获取配置目录
