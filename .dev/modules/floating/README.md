# Floating Capsule 模块

> EchoVoice 悬浮胶囊（桌面端录音指示器）

## 概述

Floating Capsule 是 EchoVoice 桌面端的可视化录音反馈组件。在用户按下热键时，在屏幕任务栏上方显示胶囊形悬浮窗，提供直观的录音状态反馈。

## 特性

- 🎤 **实时波纹动画** - 可视化音量大小
- 📍 **固定位置** - 任务栏上方，不遮挡内容
- ⚡ **快速响应** - 显示/隐藏无延迟
- 🎨 **状态指示** - 录音中/处理中/成功/失败
