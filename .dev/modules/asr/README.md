# ASR 模块

> 语音识别模块 - 负责将音频转换为文本

---

## 职责

使用 Whisper 模型进行本地语音识别，将麦克风采集的音频转换为文本。

---

## 设计文档

- [详细设计](./design.md) - 架构、流程、技术细节
- [接口定义](./interface.md) - API 规范

---

## 关键特性

| 特性 | 说明 |
|------|------|
| 模型 | Whisper.cpp (GGML 格式) |
| 语言 | 中文（当前硬编码）|
| 输入 | f32 PCM 音频 |
| 输出 | UTF-8 文本 |

---

## 依赖

- `whisper-rs` - Whisper Rust 绑定
