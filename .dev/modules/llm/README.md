# LLM 模块

> 文本润色模块 - 负责将 ASR 输出进行智能润色

---

## 职责

使用本地 LLM 对识别文本进行润色、结构化处理，提升文本质量。

---

## 设计文档

- [详细设计](./design.md) - 架构、流程、技术细节
- [接口定义](./interface.md) - API 规范

---

## 关键特性

| 特性 | 说明 |
|------|------|
| 模型 | SmolLM2 (GGUF 格式) |
| 上下文 | 512 tokens |
| 温度 | 0.7 |
| 提示词 | ChatML 格式 |
